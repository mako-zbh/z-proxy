# ZProxy - 智能代理管理器

一款功能强大的 Chrome 浏览器代理插件，提供现代化的 UI 界面和完善的代理管理功能。

## 功能特性

### 核心功能
- **代理管理** - 支持添加、编辑、删除多个代理服务器
- **一键切换** - 快速启用/禁用代理，简单方便
- **代理测试** - 测试代理连接是否正常
- **智能白名单** - 支持白名单/黑名单两种模式
- **状态指示** - 实时显示代理连接状态

### 代理类型支持
- HTTP
- HTTPS
- SOCKS5

### 白名单功能
- **白名单模式** - 仅列表中的域名使用代理
- **黑名单模式** - 列表中的域名不使用代理
- 支持通配符域名匹配

## 安装方法

### 1. 生成图标文件

在浏览器中打开 `icon-generator.html` 文件，点击"下载所有图标"按钮，将下载的图标文件放入 `icons` 目录。

### 2. 加载插件到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 zproxy 项目文件夹
6. 插件安装完成！

## 使用指南

### 添加代理

1. 点击插件图标打开弹窗
2. 点击"添加代理"按钮
3. 填写代理信息：
   - 代理名称：方便识别的名称
   - 服务器地址：IP 地址或域名
   - 端口：代理服务器端口
   - 代理类型：HTTP/HTTPS/SOCKS5
   - 用户名/密码（可选）：如果代理需要认证
4. 点击"保存代理"

### 启用代理

1. 在代理列表中选择要使用的代理
2. 点击顶部的开关按钮启用代理
3. 当开关变为绿色时，代理已启用

### 配置白名单

1. 切换到"白名单"标签页
2. 选择模式：
   - **白名单模式**：只有列表中的域名使用代理
   - **黑名单模式**：列表中的域名不使用代理
3. 在输入框中输入域名（如 google.com）
4. 点击"添加"按钮

### 测试代理

在代理列表中点击"测试"按钮，可以测试代理连接是否正常。

## 技术栈

- **Manifest V3** - Chrome 最新插件标准
- **原生 JavaScript** - 无依赖，轻量高效
- **Chrome Extension APIs** - 代理、存储、标签页管理

## 隐私说明

ZProxy 完全在本地运行，不收集任何用户数据：
- 所有代理配置保存在本地浏览器存储中
- 不会向任何第三方服务器发送数据
- 不会追踪用户浏览行为

## 安全建议

1. 仅使用可信的代理服务器
2. 定期更新代理密码
3. 不要在公共场所共享代理配置
4. 使用 HTTPS 代理保护数据传输
